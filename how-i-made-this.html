How I Made This App

By Ollie

🎮 What is Teleporting Maths?

Teleporting Maths is a game I created to help me practice my times tables and division. You control a character who collects coins and enters portals to answer maths questions!

💡 Why I Made This

I wanted to make learning times tables fun. I love coding and I thought it would be fun to have a game where you can earn coins, walk through portals and buy items for your character.

🛠️ How I Built It

I used a technique called Vibe Coding with a tool called Amazon Q Developer inside Visual Studio Code (a Microsoft coding environment). This means I could describe my requirements (what I wanted the game to do), and Amazon Q helped me write the code! This is called human-in-the-loop coding - I stay in control and make all the decisions, while Amazon Q helps me write the code faster.

I used three main technologies to build this game:

  • HTML - This creates the structure of the game
  • CSS - This makes everything look colorful and nice
  • JavaScript - This makes the game work and move

🐛 Fixing Bugs

Making a game isn't always easy - I had to fix lots of bugs! A bug is when something in the code doesn't work the way you want it to.

Example Bug: When I went to answer a maths question, I could see the cursor blinking in the answer box, but when I typed numbers, nothing appeared! I had to click on the box first before I could type.

How I Fixed It: The problem was that the game was still listening for keyboard controls (like WASD to move the character) even when I was supposed to be typing my answer. I had to tell the code to stop listening for movement keys when the question screen was showing, so the keyboard would only type into the answer box.

🎨 The Design Process

First, I described what I wanted the game to look like - these descriptions are called requirements. I came up with a wireframe (a basic layout of the game), and then we iterated from there. Iterating means I kept editing and adding more features to make the game better!

I started with the basic character and portals, then added coins, a shop system, customizable items like hats and capes, and even a flying chair!
